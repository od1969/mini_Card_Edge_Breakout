# mini Card Edge Breakout
ノートPCやNUCに内蔵できるカードエッジ状のユニバーサル基板です。  
mini PCI express互換！  
  
以下の信号線を引き出すことができます。  
 3.3V  
 1.5V  
 3.3V(AUX)  
 GND  
 LED PLAN  
 LED WLAN  
 LED WWAN  
 USB D+  
 USB D-  
 SMBus CLK  
 SMBus DAT  
 W_DISABLE#  
 PERST#
